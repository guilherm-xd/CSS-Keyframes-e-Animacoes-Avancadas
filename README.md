# Motion Studio — Interfaces em Movimento

Uma landing page moderna e criativa desenvolvida apenas com HTML e CSS, focada em animações avançadas, efeitos visuais e microinterações. O projeto explora o uso intensivo de @keyframes para criar uma experiência visual dinâmica sem JavaScript.

---

## Sobre o Projeto

Este projeto consiste em uma landing page de um estúdio fictício chamado Motion Studio, focado em design, animações e interfaces modernas.

A proposta foi criar uma interface com aparência premium, inspirada em sites de estúdios criativos e páginas de apresentação de serviços digitais, explorando ao máximo efeitos visuais em CSS.

O objetivo principal foi praticar animações avançadas utilizando @keyframes, além de desenvolver uma landing page responsiva e visualmente impactante sem frameworks ou JavaScript.

---

## Funcionalidades

- Loader animado de entrada
- Header fixo com efeito de transparência (glass)
- Hero section com destaque tipográfico e animação neon
- Botões com efeito pulsante e hover animado
- Cards interativos na seção de serviços
- Carrossel automático 100% CSS
- Galeria com mini animações (glitch, órbita e efeitos visuais)
- Hover animado em cards e botões
- Backgrounds com gradientes e iluminação dinâmica
- Layout responsivo

---

## Tecnologias Utilizadas

- HTML
- CSS

---

## Animações Utilizadas

O projeto utiliza diversas animações avançadas feitas com @keyframes, incluindo:

- translateY() para efeitos de flutuação
- translateX() para movimentação do carrossel
- scale() para pulsação e interações
- rotate() para loader e órbitas
- efeitos de neon com text-shadow
- animações de glitch com camadas sobrepostas
- animações de background dinâmico
- hover com elevação e transição suave
